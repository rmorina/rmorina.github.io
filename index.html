Google - Software Engineering Intern, Accessibility (Summer 2015)

I worked on improving the efficiency of Switch Access, an accessibility service that helps users with motor disabilities access their Android devices.

Switch Access enables a user to control an Android device using a single or multiple switches. Using these switches, a user navigates through any screen of selectable items. Currently there are two basic methods of navigation: linear and row-column scanning. In linear scanning each of the items in the screen is highlighted one after the other and once the desired item has been highlighted the user presses the switch to choose that item. In row-column scanning each individual row is highlighted first and once the user chooses a desired row, each columns is highlighted until the user chooses the desired item.

I developed two new scanning features that were in particular geared towards improving the typing efficiency for these users: (1) option scanning and (2) probability-based scanning. In option scanning, given the N switches a user may choose to use, the items in the screen are divided into N buckets. Each group is highlighted in colors that correspond to the user's switches. The user then recursively selects the desired item. Probability scanning was a novel idea that I experiemented with. I utilized an n-gram character model to obtain the probabilities of each letter. These probabilities were then used to construct Huffman trees that guaranteed an optimal encoding for each letter. This meant that at any point in time we were able to optimize the number of clicks a user would have to make to select the desired option.

In September 2015, Switch Access went open sourced. Here is the code base. I have authored HuffmanTreeBuilder.java, PPMTrie.java, ProbabilityModelReader.java, and TalkBackOrderNDegreeTreeBuilder.java.

I have since grown to be a passionate advocate of Accessibility. In November 2015 and April 2016, I gave a lecture - Data Compression with Huffman Encoding - to about 400 students. I primarily focused on the application of Huffman Encoding in improving the typing efficiency for users with motor impairment. I developed a demo of a typing interface to show the efficiency improvement when using Huffman Encoding.

Linux-like kernel

As part of Carnegie Mellon's Operating System class, working with a partner I designed and implemented a linux-like kernel for the x86 architecture.

The kernel supports basic functionalities such as preemtive multi-process scheduling in a round robin fashion, process spawning, two level page table VM, and a simple shell. The kernel also supports device drivers for the keyboard, the console, and the timer.

Furthermore, as part of the project preceeding the implementation of the kernel, I developed a thread library and concurrency primitives, which included mutexes, condition variables, semaphores, and read/write locks. The kernel was then able to load and run user space programs that used my implementation of the thread library and the concurrency primitives.

Since this is an ongoing class at CMU, the code is not public, but I can provide it on request.

EchoExplorer - An echolocation training platform

EchoExplorer is a research project that I've been involved with since August 2014. The aim of this research project is to understand how the auditory and visual channels can work together to enhance visual capabilities for individuals with limited vision. Inspired by the exceptional echolocator, Danish Kish, we are in particular focusing on understanding how echoes can convey visual information and how humans can learn to pick up these auditory cues.

I have been focusing on two main parts of the research: (1) artificial echo generation, and (2) training app development. The artificailly generated echoes are integrated into the training app to implement tasks that aim to teach the users how to understand the auditory cues. The echo generation code is currently done in Matlab, while the app is developed on the Android platform.

The research work I've done has been supported by the CSoI (Center for Science of Information) Channels Program, an NSF program that provides undergraduate students with the opportunity to participate in a research project related to the Science of Information. Mechanical Logic

The aim of this project was to demystify basic Boolean algebra concepts through the use of mechanical logic gates as more intuitive teaching tools. Over a course of a year, my team and I went through a couple of design iteration and were ultimately able to use these gates at various outreach events to teach middle and high school kids the basics of Boolean algebra.

For CMU's Build18 hack week in 2013, we developed very small, LEGO-compatible version of the gates. We won the First Penguin award, named after a term coined by CMU's professor Randy Pausch, for a project which deviated most significantly from the theme of the electronics- and programming-themed competition.
